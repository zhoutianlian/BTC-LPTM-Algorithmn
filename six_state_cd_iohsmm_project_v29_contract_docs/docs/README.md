1. `ALGORITHM_PRINCIPLES.md` — 六状态清算—吸收市场状态引擎的算法原理说明。
2. `FEATURE_CONTRACT.md` — 输入特征契约：required files、核心列、金融含义与 no-leakage 约束。
3. `OUTPUT_CONTRACT.md` — 输出契约：输出文件、字段语义、策略接入和误用风险。
4. `EXECUTION_MANUAL.md` — 执行手册。
5. `CODE_DESIGN.md` — 工程结构和模块职责。

HTML 版本用于浏览器阅读；Markdown 版本用于版本管理与协作编辑。
